Word Highlighter

Word Highlighter Chrome Extension

Highlight and navigate through specific words on any webpage with real-time word counting and dark mode support.

Features

Real-time Highlighting

Instantly highlight words as you type with real-time word counting and colorful visual feedback.

Easy Navigation

Navigate between highlights with Previous/Next buttons and smooth scrolling animations.

Dark Mode

Toggle between light and dark modes for comfortable viewing with optimized color schemes.

Word Statistics

Real-time word counting and statistics for each highlighted term across the entire page.

LinkedIn Optimized

Specially optimized for LinkedIn with support for collapsed job ads and dynamic content.

Multi-Color Highlights

Each word gets its own vibrant color from a palette of 10 distinct, easily readable colors.
